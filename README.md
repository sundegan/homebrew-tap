# Sundegan Homebrew Tap

Homebrew tap for Sundegan apps and tools.

## Install Json Studio

```sh
brew install --cask sundegan/tap/json-studio
```

## Install macOS Icon Generator

```sh
brew install sundegan/tap/macos-icon-generator
```

## Available Formulae

- `macos-icon-generator` - Generate macOS-style icons from square SVG, PNG, or JPEG images.

## Available Casks

- `json-studio` - Fast, local-first JSON workspace for everyday development.
